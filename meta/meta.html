what is metadata?

Metadata is "data that provides information about other data". In other words, it is "data about data." The use of metadata on web pages can be very important. It contains descriptions of the page’s contents, as well as keywords linked to the content. This metadata is often displayed in search results by search engines, making its accuracy and details very important since it can determine whether a user decides to visit the site or not. This information is usually expressed in the form of meta tags. Meta tags are often evaluated by search engines to help decide a web page’s relevance.

  • Meta Keywords Attribute - A series of keywords you deem relevant to the page in question. Title Tag - This is the text you'll see at the top of your browser. Search engines view this text as the "title" of your page. Meta Description Attribute - A brief description of the page. Meta Robots Attribute - An indication to search engine crawlers (robots or "bots") as to what they should do with the page.
